⚙ 구글 시트 클라우드 동기화 설정

☁️ 실시간 동기화 및 50인 공유 연동 가이드

이 예약 웹앱은 정적 페이지이므로, 다른 교사 노트북과 실시간으로 예약을 공유하려면 무료 구글 스프레드시트를 연동해야 합니다. 아래 단계를 따라해 주세요.

  1. 구글 시트 생성: 구글 드라이브에서 새 구글 스프레드시트를 하나 만듭니다.
  2. 스크립트 창 열기: 시트 상단 메뉴에서 [확장 프로그램] > [Apps Script]를 클릭합니다.
  3. 코드 붙여넣기: 기본 생성된 코드를 다 지우고, 아래의 소스코드를 복사하여 붙여넣은 뒤 저장(💾)합니다.
  1. 배포 실행: Apps Script 창 우측 상단의 [배포] > [새 배포]를 누릅니다.
  2. 유형 선택: 톱니바퀴 아이콘을 누르고 [웹 앱]을 선택합니다.
  3. 권한 설정: 실행할 사용자: 나(교사 본인 계정), 액세스할 수 있는 사용자: 모든 사용자(Anyone)로 설정한 뒤 배포 버튼을 누릅니다. (액세스 권한 승인 창이 뜨면 허용해 줍니다.)
  4. URL 복사: 완료 화면에 나타나는 웹 앱 URL 주소를 복사합니다.
